# Liberu Browser Game (PBBG)
## Product Feature & Design Scope

> **Version:** 1.0  
> **Framework:** Laravel 13 + Filament 5  
> **Frontend:** Livewire 4 + Alpine.js (+ optional Three.js enhancements)  
> **PHP:** 8.5  
> **Genre:** Persistent Browser-Based Multiplayer RPG (PBBG)

---

# Vision

Liberu Browser Game is a modern Persistent Browser-Based Game (PBBG) designed as a multiplayer fantasy RPG with deep progression, player interaction and long-term engagement.

Rather than relying solely on static text, the game should leverage modern web technologies to create an immersive experience using Livewire, Alpine.js and optional HTML5/Three.js enhancements while remaining accessible from any modern browser without requiring downloads.

The game should be enjoyable for both casual and competitive players while providing optional premium subscriptions that enhance convenience without becoming pay-to-win.

---
